# TasteDrop – Online Food Ordering Frontend

A multi-page restaurant menu and cart system built with HTML, CSS, Bootstrap 5, and vanilla JavaScript.

## Pages
- **Home** – Hero, How It Works, Popular Dishes (from JSON), Testimonials, Footer
- **Menu** – Category tabs, search, sort by price, quantity selector, add to cart
- **Cart** – Cart table, quantity controls, remove button, order summary

## Features
- Dark mode with localStorage persistence
- Cart stored in localStorage (survives page reload)
- Promo codes (TASTE10 = 10% off, FREEDEL = free delivery)
- Checkout form with validation

## How to Run
1. Clone the repository
2. Open with Live Server in VS Code
3. Start from `index.html`

## Technologies
- Bootstrap 5.0.2
- Font Awesome 6
- Google Fonts (Poppins)
- LocalStorage API
